English:
Use 7-Zip's dynamic-link library 7z.dll to read and decompress archives.
7z.dll can be obtained from the 7-Zip executable software.
If this program is compiled as 32-bit, please use the 32-bit "7z.dll"; 
if compiled as 64-bit, please use the 64-bit "7z.dll".

7-Zip Official: https://www.7-zip.org/
7-Zip Download: https://www.7-zip.org/download.html

7za.dll or 7zxa.dll from the SDK only handle the 7z format and cannot decompress other archive formats.

When using 7-Zip's dynamic-link library, please adhere to 7-Zip's relevant usage guidelines.

https://www.7-zip.org/faq.html
https://www.7-zip.org/sdk.html
